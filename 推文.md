# talk-mbti 推文

## 推文（微博/X/朋友圈）

做了一个不一样的 MBTI 测试工具 talk-mbti。

市面上的 MBTI 产品本质上都是**问卷的皮**，几十道选择题做下来，体验像在参加高考。问题是，人在「选 A 还是选 B」时调用的是理性自我，跟真实性格之间隔了一层滤镜。

talk-mbti 的做法是：**直接跟你聊天。**

通过 8-12 轮自然对话，从你说话的方式、面对场景的第一反应、做选择时的底层逻辑里提取行为信号，最后给出一份引用你原话的人格分析报告。

几个设计上的关键点：

1️⃣ **三阶段对话引擎**：破冰暖场 → 深度追问 → 校准收尾。前半段你以为在跟朋友闲聊，后半段 AI 已经在交叉验证你的矛盾信号了

2️⃣ **动态评分**：每轮对话实时提取信号并加权打分，下一个问题自动瞄准置信度最低的维度。不是固定题库，是活的对话流

3️⃣ **诚实标注**：如果某个维度的信号不够充分，报告里会直接说「这个维度我没太大把握」，而不是硬凑一个结果糊弄你

当前是 Phase 1，做成了 Claude Code / Codex 的 Skill，终端里输入 `/mbti-analysis` 就能开始。Phase 2 会做成 Web 应用，用 Gemini 做对话引擎。

**MBTI 本身有没有科学性可以争论，但「通过对话而非问卷来了解一个人」这件事，底层逻辑是对的。**
